# 亲密关系

>- 两个人在一起谈恋爱，就应该是真诚的，大大方方的，坦坦荡荡的，毫无顾忌。两个人不需要畏手畏脚，
> 爱就要大大方方地表达。喜欢什么，要说出来，不开心也要说出来，讨厌什么，更要说出来，流泪了，说说为什么；吃醋生闷气，也要说说原因。
> 
>- 你可以胆小，可以不好意思，但绝对不可以别扭。不是什么事都可以一直拖着靠时间来解决。恋爱的你们没有上帝视角，
> 爱和委屈都要大声说出来，不要互相猜来猜去，不要口是心非。相爱的时间太宝贵了，不能在奇奇怪怪的问题上消磨爱意。
> 恋爱的意义在于：彼此弥补不足，成为对方的避风港和精神寄托。让你们在每个情绪低落的夜里，都能有个依靠，互相救赎，成为对方的靠山。
> 真正的爱，就应该是这样的。
> 
>- 要学会大大方方的表达爱意。爱不是冷冰冰的，而是炙热的，永远不要做做那种说反话的笨蛋，把喜欢的人越推越远。
