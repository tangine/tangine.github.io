# __申论__

## __考查能力__
1. __政策理论素养__: 考生需掌握国家政策法规，了解社会热点问题，能够结合实际进行分析
2. __综合分析能力__: 需要对给定资料进行全面理解和分析，提炼出核心观点和问题
3. __文字表达能力__: 在命题作文中，考生需清晰表达思想，逻辑严谨，语言通顺

## __如何精读人民日报__
1. __通读全文__: 初步了解文章基本内容，这一步粗略读，要快。重点扫视首段、尾段，以及每段话的首句、尾句，快速定位文章整体结构。 
2. __分析结构__: 段首一般为主旨大意，所有段首串起来看，分析结构和思路。 
3. __逐段精读__: 论据、论证方法积累借鉴。 
4. __亮点积累__: 精彩的表达，万能句式结构、素材。 
5. __化为己用__: 复述一遍文章结构，写感悟，即可加深对文章的理解，又可通过动笔，逐步提升写作水平和深度，还可加深记忆，避免看过就忘。
